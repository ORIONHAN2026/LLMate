功能特性

为什么选择 LLMWork？

不只是聊天客户端，更是你的 AI 生产力工作台。从多模型管理到文件操作，从 MCP 扩展到技能系统，一站式满足所有 AI 工作需求。

🖥️ 多模型统一管理

一个界面管理 OpenAI、Claude、Gemini、DeepSeek、通义千问、智谱、Ollama 等 8+ 主流 AI 模型。独立配置 API Key、Temperature、System Prompt，自由切换。

  • 单个模型独立配置，互不干扰
  • 支持在线模型 API + 本地 Ollama 模型
  • 模型选择器一键切换，无需重启对话

🔌 MCP 协议原生支持

完整支持 **Model Context Protocol (MCP)**，内置 MCP 市场一键安装。

  • 支持 SSE、HTTP、Streamable HTTP 三种传输协议
  • Stdio 本地进程 + URL 远程连接两种方式
  • 200+ 第三方 MCP 服务可用
  • 搜索、文件管理、数据库、浏览器自动化…

📄 AI 文件操作引擎

让 AI 直接创建和编辑 Word、Excel、PDF、PPT 文档，处理图片、执行代码。

  • Word / Excel / PDF / PPT 文档生成与编辑
  • Python、Node.js、Shell 运行时环境
  • 图片处理与 OCR 识别
  • Markdown 文档渲染与导出

⭐ 技能系统 & 市场

创建自定义 AI 技能角色，内置技能市场提供预置模板。

  • 代码审查、内容创作、数据分析等模板
  • 一键绑定到模型和会话
  • 自定义 System Prompt 与角色设定
  • 让 AI 成为真正的领域专家

🧠 智能记忆压缩

长对话自动记忆压缩，突破上下文长度限制。

  • 自动检测上下文窗口使用量
  • 可视化展示压缩记忆和对话轮次
  • 在超长对话中保持连贯理解

⏰ 定时任务 & 快捷指令

  • Cron 定时任务，到时间自动发送预设消息
  • 模型级和会话级快捷指令
  • 一键发送常用提示词
  • 大幅提升日常工作效率
