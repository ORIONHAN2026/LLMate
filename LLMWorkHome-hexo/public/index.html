核心功能

🖥️ 多模型统一管理

一个界面管理 OpenAI、Claude、Gemini、DeepSeek、通义千问、智谱、Ollama 等 8+ 主流 AI 模型。独立配置 API Key、Temperature、System Prompt，自由切换。

🔌 MCP 协议原生支持

完整支持 Model Context Protocol，内置 MCP 市场一键安装。支持 SSE、HTTP、Streamable HTTP 传输，Stdio 和 URL 双连接方式，无限扩展 AI 能力边界。

📄 AI 文件操作引擎

让 AI 直接创建和编辑 Word、Excel、PDF、PPT 文档，处理图片、执行代码。内置 Python、Node.js、Shell 运行环境，AI 帮你完成实际工作。

⭐ 技能系统 & 市场

创建自定义 AI 技能角色，内置技能市场提供代码审查、内容创作、数据分析等预置模板。一键绑定到模型和会话，让 AI 成为领域专家。

🧠 智能记忆压缩

长对话自动记忆压缩，突破上下文长度限制。可视化展示压缩记忆和对话轮次，让 AI 在超长对话中始终保持连贯理解。

⏰ 定时任务 & 快捷指令

支持 Cron 定时任务，到时间自动发送预设消息。模型级和会话级快捷指令，一键发送常用提示词，大幅提升工作效率。
