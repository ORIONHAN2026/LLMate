## 核心功能

### 🚀 可视化安装，零技术门槛

提供可视化安装向导，全程图形化操作，无需任何技术人员即可完成部署和配置。让每一位企业管理者都能轻松上手，自主进行企业 AI 资源管理与调度。

### 🖥️ 多模型统一管理

支持接入 DeepSeek、OpenAI、Google Gemini、阿里云百炼、腾讯云 TokenHub 等主流大模型供应商，企业可在一个平台上集中管理所有 AI 模型资源。统一配置 API 地址、安全密钥、模型参数，随时按需切换。

### 🏢 按分组的会话管理

可按团队、部门或项目创建会话分组，每个会话独立分配请求地址、安全密钥、用量配额和访问权限，实现大模型资源的精细化按需按量分配。实时统计每个会话的调用量与费用，让 AI 资源使用透明可控。

### 🔌 内置 MCP 配置，资源可控访问

原生支持 Model Context Protocol，内置 MCP 工具配置能力。管理员可统一管理企业资源的 MCP 接入权限，实现文件系统、数据库、内部 API 等企业资源的可控访问，确保数据安全合规。

### 📝 企业级系统提示词

支持管理员设置企业全局系统提示词，统一规范 AI 输出行为。从源头杜绝违规内容，确保 AI 对话始终符合企业合规要求和品牌调性。支持按会话、按场景配置差异化提示词策略。

### 🔓 免授权便捷访问

支持免授权访问模式，企业内部用户无需复杂的密钥配置即可快速使用 AI 服务。兼顾用户体验与管理安全，降低企业 AI 推广使用门槛。
