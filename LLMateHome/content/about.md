## LLMate — AI 生产力工作台

LLMate 是一款面向企业的智能 AI 工作台，提供可视化的多模型管理、MCP 服务编排、会话用量管控等核心能力，帮助企业安全、高效地落地 AI。

### 核心价值

- **零技术门槛** — 可视化安装与配置，无需技术人员即可部署
- **精细化管控** — 按会话、按分组分配 AI 资源与用量配额
- **安全合规** — 企业级系统提示词、内容审计、权限控制
- **开放生态** — 原生 MCP 协议支持，可接入文件系统、数据库、内部 API 等外部工具
