会话的基础信息用于标识与归类该会话，方便按员工或组织进行区分与管理。在**会话设置**页的 **基础信息** 区域可进行以下查看与编辑。

### 可编辑项

- **会话名称**：双击即可修改，便于按员工或组织识别（如"技术部-后端开发"）
- **所属分组**：双击即可修改，如"技术部""市场部""财务部"，用于会话归类

{{< figure src="images/session-basic-info/step-edit.png" caption="双击会话名称或所属分组即可编辑" >}}

### 只读展示项

- **绑定模型**：展示当前会话绑定的模型名称与具体标识（在创建会话或模型管理中设定）
- **关联提示词**：若该会话的模型或连接方式带有预设提示词，会在此展示
- **消息数量**：当前会话累计的消息条数

{{< figure src="images/session-basic-info/step-readonly.png" caption="会话基础信息 - 绑定模型、关联提示词、消息数量（只读）" >}}

> 基础信息会随会话使用实时更新；名称与分组建议结合企业实际的员工 / 组织维度统一规划，便于后续检索与权限分配。
