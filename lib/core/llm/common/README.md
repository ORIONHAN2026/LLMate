# core/llm/common/ - 公共提示词与消息构建

## 文件说明

| 文件 | 行数 | 说明 |
|------|------|------|
| `system_prompts.dart` | 556 | 所有系统提示词常量：核心行为规则、反幻觉指令、工具使用规则、语言设置、深度思考模式、各工作模式专用提示词 |
| `message_builder.dart` | 85 | 消息构建器：从模型配置、技能提示词、MCP 提示词、模式提示词组装系统消息列表 |

## 提示词体系

系统提示词按层级组装：

1. **模型配置提示词** — 用户自定义的 System Prompt
2. **技能提示词** — 当前绑定的 Skill 内容
3. **MCP 提示词** — 当前绑定的 MCP 工具说明
4. **模式提示词** — 当前工作模式的专用指令
5. **核心规则** — 通用行为约束（语言、格式、安全等）
