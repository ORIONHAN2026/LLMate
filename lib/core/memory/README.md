# core/memory/ - 对话记忆压缩

## 职责

通过 LLM 对长对话历史进行智能压缩，生成简洁的上下文摘要，避免 token 溢出。

## 文件说明

| 文件 | 行数 | 说明 |
|------|------|------|
| `memory_compressor.dart` | 80 | 记忆压缩器：将已压缩摘要 + 近期对话发送给 LLM 生成新的精简摘要 |

## 压缩流程

```
1. 读取已有的压缩记忆 (memory.md)
2. 获取最近的对话轮次 (MemoryTurn[])
3. 构建压缩提示词：摘要 + 近期对话
4. 调用 LLM 生成新摘要
5. 将新摘要写回 memory.md
```

## 触发时机

- 会话消息数超过阈值时自动触发
- 由 `SessionController` 异步调度
