# features/chat/widgets/sidebars/ - 侧边栏组件

## 职责

聊天界面的左/右侧边栏，按工作模式展示不同内容。

## 文件说明

| 文件 | 行数 | 说明 |
|------|------|------|
| `chat_left_sidebar.dart` | 962 | **左侧边栏**：会话列表、搜索、收藏、重命名、删除、新建、工作模式图标 |
| `chat_right_sidebar.dart` | 1633 | **右侧边栏**：会话记忆/产物、文件树浏览器、模式标签页（合同/发票/聊天室/创作） |
| `contract_sidebar.dart` | 476 | 合同模式面板：合同要点、审查流程、争议点、笔记 |
| `invoice_sidebar.dart` | 211 | 发票模式面板：发票摘要、明细、报销记录、笔记 |
| `chatroom_sidebar.dart` | 336 | 聊天室模式面板：角色列表、笔记 |
| `mindmap_widget.dart` | 147 | 脑图渲染：可折叠的树形结构，颜色编码深度层级 |
