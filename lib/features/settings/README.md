# features/settings/ - 设置模块

## 职责

应用全局设置的管理界面。

## 目录结构

```
settings/
└── pages/
    ├── settings_page.dart           # 设置主页：左侧导航 + 右侧内容区
    ├── modelssetting.dart           # 模型管理设置：列表/添加/编辑/删除
    └── other_settings_page.dart     # 其他设置：语言/主题/工作模式
```

## 设置项

| 标签页 | 内容 |
|--------|------|
| 模型管理 | LLM 模型列表、添加在线模型、编辑配置 |
| 连接器管理 | MCP 服务配置 |
| 技能管理 | 技能列表、创建、编辑 |
| 其他设置 | 语言切换、主题选择、工作模式 |
